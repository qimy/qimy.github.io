文章目录

今天盘算了一下这博客要修修补补的地方，发现我不是处女座也没有强迫症，怎么事情就是那么折腾呢？清单123就放结尾好了，开始正题吧<(￣︶￣)↗[GO!]

pacman的字体可以分为两种，一种是header和footer的字体样式。
一种就是剩下的字，微软雅黑啦，Helvetica Neue啦，可以随便设。

而header和footer是英文字体。正常情况下是这样子的。

一旦换成中文，就变成了这个样子。

咦⊙▽⊙？
高大上的艺术字怎么变成了这样呢( -‘`-)？
不要啊/(ㄒoㄒ)/~~

so，换字体吧╮(￣▽￣”)╭ 。

查看hexo\themes\pacman\config.yml

#### Font
ShowCustomFont: true
## you can change custom font in `variable.styl` and `font.styl`which in the theme folder `/pacman/source/css`.

找到variable.styl。查看其中跟字体有关的。

//Font  
font-default = "Helvetica Neue", "Helvetica","Microsoft YaHei", "WenQuanYi Micro Hei",Arial, sans-serif  
font-serif = "Georgia", serif  
font-mono = Monaco, Menlo, Consolas, Courier New, monospace
font-custom-family = "covered_by_your_graceregular"
font-custom-filename = coveredbyyourgrace-webfont  
font-icon-family = "FontAwesome"  
font-icon-filename = fontawesome-webfont  
font-icon-version = "4.0.3"  
ShowCustomFont = hexo-config("ShowCustomFont")  
font-size = 100%  
line-height = 1.5

以上这段我实在网上粘的，因为本体已经被修改的面目全非了233。
恩，高亮代码还木有学会，所以先这样看吧(☄⊙ω⊙)☄。

font-default是页面上用到的字体，按先后顺序排列。可以把喜欢的字体放在前面，也可以任意更换。注意,后面有空格。
font-mono貌似是代码字体，先后顺序排列。同上。

下面这两行就是要改的header和footer的字体了。

font-custom-family = "covered_by_your_graceregular"
font-custom-filename = coveredbyyourgrace-webfont

仔细一看，covered by your graceregular？什么鬼( ｀д′ )？

总之呢，这个字体不能那么简单的说换就换。
我们就按照步骤开始吧(ง •_•)ง。

下载叶根友简书繁字体

上传fontsqirrel网站
按下图点选选项。

点击download your kit 下载。

得到一个.zip的压缩包。解压后将.eot .svg .ttf 以及.woff统统塞到hexo\themes\pacman\source\font文件夹里。

打开hexo\themes\pacman\source\css\_base\variable.styl
将下面两行里的coveredbyyourgrace修改为刚刚塞进去的那几个文件名。我的是ygyxsf。

font-custom-family = "covered_by_your_graceregular"
font-custom-filename = coveredbyyourgrace-webfont

改为

font-custom-family = "ygyxsf"
font-custom-filename = ygyxsf-webfont

然后
